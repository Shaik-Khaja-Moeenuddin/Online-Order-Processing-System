# Online Order Processing and Tracking System

## Description
A Python-based application to manage online orders and track delivery status using object-oriented and modular programming.

## Features
- Place Order
- View Orders
- Update Status
- Track Orders
- JSON Data Storage

## Technologies Used
- Python
- OOP Concepts
- Modules
- File Handling (JSON)

## Project Structure
models/ → order, record, tracking  
utils/ → validators  
storage/ → json handling  
main.py → main program
